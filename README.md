# ANTLR4_Interpritator

Проект представляет собой интерпритатор смешанного языка программирования.

Проект создан в учебных целях.

## Уже реализованно

### Поддержка объектов

```Java
 class ClassName {
    
    fun1(params){
        return params;
    } 
 }
 
 str1 = "Строка1"
 tmp1 = ClassName.fun1(str1);
 print(tmp1);
```
>####"Строка1"

### Цикл forearch

```Java
a = 0;
forearch(a,3){
   print(a);
}
```
>####0

>####1

>####2
